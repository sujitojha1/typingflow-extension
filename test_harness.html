Machine Learning

Backpropagation is an algorithm used to calculate the gradients of the loss function with respect to the weights in an artificial neural network. It is widely used for training deep neural networks and remains foundational for modern AI models.

The term abbreviation for "backward propagation of errors" refers to the fact that the gradient is computed back-to-front, starting from the output layer through to the input layer.

An artificial neural network consists of layers of interconnected nodes. These mathematical constructs loosely simulate biological neurons in animal brains, receiving inputs, applying weights, and emitting signals.
